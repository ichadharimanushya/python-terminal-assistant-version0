A terminal based simple assistant using python (first project)

Features:
- Multi-User support
- Add/delete notes
- Todo management
- Statistics
- Other utilities as such
(
current date and time,
calculator,
timer,
random joke, facts, quotes,
weather information
)

Technologies used:
- Python
- Object-Oriented Programming (OOP)
- JSON for data storage
- File handling
- APIs using urllib
- Classes and methods
- Error handling

Project structure:
- main.py
- commands.py
- data.json
- notes.txt

How to run:
python main.py
Enter a username when prompted. If user does not exist, a new user will be created automatically.

python commands.py
contains many methods in class Assistant such as:
- time
- date
- todo
- todos
- stats
- help
- etc...
